# 2026 日本家庭旅行三方案

這是一個給家人討論 2026/10/30 到 2026/12/02 日本旅行用的單頁靜態網站。
同一個網址可以切換三套 34 天、33 晚的逐日行程。

## 內容

- 方案 1：函館到福岡的完整環日本壯遊版
- 方案 2：東京、金澤、高山、京都、廣島、福岡的楓葉平衡版
- 方案 3：東京、關西、福岡三大區域孝親慢遊版
- 大字、高對比、城市分段與體力標記，適合爸媽用手機或平板閱讀
- `#plan1`、`#plan2`、`#plan3` 可直接分享指定方案
- 可篩選賞楓日、移動日與休息日
- 本機瀏覽器互動：分方案留言、列印、複製家庭討論摘要

## 使用

執行 `node tests/static-server.js`，再開啟 `http://127.0.0.1:4175`。

測試指令：`node --test tests/itinerary.test.js`

線上版：<https://ymeroom.github.io/japan-family-trip-discussion/>
